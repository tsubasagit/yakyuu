# Changelog

## v0.3.0 (2026-04-17)

### 更新理由
ユーザーより「OBS上で両チームの打順を同時に表示したい」というご要望をいただき対応。これまではオーバーレイの打順カードが攻撃中チームのみ自動切替で表示される仕様だったため、両チームを並べて視聴者に見せたい配信シーンで不便があった。

### 新機能
- オーバーレイに両チームの打順を同時表示するトグルを追加（コントロールパネルからON/OFF）
- 両チーム表示時、各打順カードを個別にドラッグで配置可能
- 両チーム表示時、各カードは相手チームの登板中投手を独立して表示

### 既存挙動への影響
- トグルはデフォルトOFF。従来通り攻撃中チームのみ表示する挙動を維持
- トグルONに切り替えると両チームのカードが個別ドラッグ可能になる

## v0.2.0 (2026-04-04)

ブラウザ表示での動作が安定。OBSウインドウキャプチャで「得点スコア」「打順・選手」パネルを個別にキャプチャして配信利用可能。

### 新機能
- 「前の打者」ボタン（打順の巻き戻し）
- コントロールパネルのセクション並び替え機能（▲▼ボタン / ドロップダウン）
- コントロールパネル 2カラムレイアウト
- 打順パネルを両チーム同時表示
- CSVインポート機能（打順・選手データ）
- サンプルデータ（広島カープ / ソフトバンク）

### 修正
- チーム名・打順・投手のオーバーレイ反映問題を修正
- チーム名を1フィールドに統合、スコアボード R/H/E を日本語化
- サンプル読込ボタンの表記改善

### 既知の制限
- OBS ブラウザソース（BroadcastChannel API）での同期は不安定なケースあり
- OBS ウインドウキャプチャでの利用を推奨（得点スコア / 打順・選手パネル）
