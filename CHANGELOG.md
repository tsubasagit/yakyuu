# Changelog

## v0.2.0 (2026-04-04)

ブラウザ表示での動作が安定。OBSウインドウキャプチャで「得点スコア」「打順・選手」パネルを個別にキャプチャして配信利用可能。

### 新機能
- 「前の打者」ボタン（打順の巻き戻し）
- コントロールパネルのセクション並び替え機能（▲▼ボタン / ドロップダウン）
- コントロールパネル 2カラムレイアウト
- 打順パネルを両チーム同時表示
- CSVインポート機能（打順・選手データ）
- サンプルデータ（広島カープ / ソフトバンク）

### 修正
- チーム名・打順・投手のオーバーレイ反映問題を修正
- チーム名を1フィールドに統合、スコアボード R/H/E を日本語化
- サンプル読込ボタンの表記改善

### 既知の制限
- OBS ブラウザソース（BroadcastChannel API）での同期は不安定なケースあり
- OBS ウインドウキャプチャでの利用を推奨（得点スコア / 打順・選手パネル）

## v0.1.0 (2026-03-17)

初回リリース。

### 新機能
- イニング別スコアボード（9回＋延長対応）
- BSO（ボール・ストライク・アウト）カウント
- 走者ダイヤモンド表示
- 打順カード（打率・HR・打点・OPS）
- 投手情報（登板数・勝敗・投球数）
- 演出エフェクト（ホームラン・三振・チェンジ）
- テロップ・ティッカー表示
- マスコット表示
- ドラッグによるパネル位置調整
- BroadcastChannel API によるタブ間リアルタイム同期
- ランディングページ（OBSセットアップガイド付き）
- GitHub Pages デプロイ対応

### 修正
- OBS Browser Source 互換性改善
- DraggableBox 無限再レンダリング修正
- localStorage 書き込み競合修正
- BroadcastSync の位置上書き防止
