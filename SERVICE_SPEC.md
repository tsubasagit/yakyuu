# yakyuu — 野球中継スコアボードオーバーレイ

## 概要
野球の試合をライブ配信する際に、テレビ中継のようなスコアボードUIをOBSのブラウザソースとして重ねて表示するWebアプリ。

## ユーザー
- **スコアキーパー**: `/control` で試合データを手動入力
- **配信視聴者**: OBS経由で `/overlay` のスコアボードを視聴

## 機能
- イニング別スコア表示
- BSO（ボール・ストライク・アウト）カウント
- 走者ダイヤモンド表示
- 打者・投手情報
- プレーバイプレー経過ログ
- コントロールパネルからのリアルタイム操作

## 技術
- React + Vite + TypeScript + Tailwind CSS
- Zustand（状態管理 + localStorage永続化）
- BroadcastChannel API（タブ間リアルタイム同期）
- サーバー不要（静的配信のみ）

## データモデル
- GameState: チーム、イニング、カウント、走者、選手情報、経過ログ
